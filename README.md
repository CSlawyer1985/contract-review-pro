## 📖 项目简介

**Contract Review Pro** 是一款基于三观四步法等专业方法论构建的智能合同审核系统。它融合了商业思维、法律专业性和实务经验，为律师、企业法务和业务人员提供全面、专业、高效的合同审核服务。

💡 **三观四步法** - 宏观+中观+三维度，理解→设计→起草→审查四步流程
🔍 **智能风险评估** - 多维度综合评分（0-100），量化合同风险
📚 **30种合同类型** - 覆盖买卖、租赁、借款、建设工程、股权转让等常见合同
🎯 **专业水准** - 由资深律师 + AI 专家双重背景打造，兼具法律专业性和技术先进性

## 🏗️ 核心框架

### 一、三观四步法方法论

```
三观（三个层面）：
1. 宏观层面 - 交易结构
   - 交易类型选择、交易路径设计、交易主体确定、交易时序规划、风险分散布局

2. 中观层面 - 合同形式
   - 合同类型确定、合同体例设计、条款完整性检查、权利义务平衡性

3. 微观层面 - 合同条款
   - 语言表达精准性、逻辑结构一致性、形式规范性

四步（四个步骤）：
理解交易 → 设计结构 → 起草合同 → 审查完善
 (宏观)    (宏观)      (中观)      (微观)
```

### 二、三维审查法

```
维度一：商业维度 💼
- 核心问题：这笔交易从商业上是否合理？
- 理解交易本质、识别商业风险、评估商业合理性

维度二：法律维度 ⚖️
- 核心问题：从法律上是否有效、完整？
- 合法性审查、有效性审查、权利义务平衡性

维度三：实务维度 🔧
- 核心问题：在实践中是否可执行、可操作？
- 可执行性、可操作性、争议预防
```

### 三、智能风险评分系统

```
综合评分公式：
综合评分 = 商业维度(30%) + 法律维度(40%) + 务实维度(30%)

风险等级：
- ≥80分：高风险 - 必须修改关键条款
- 60-79分：中等风险 - 建议优化重要条款
- 40-59分：低风险 - 可考虑完善一般条款
- <40分：极低风险 - 可直接签署
```

### 使用方式一：快速审核

**步骤 1：准备合同文本**

将合同文本准备好，可以是字符串或从文件读取。

**步骤 2：调用审核系统**

```python
from main import quick_review

contract_text = """
买卖合同

甲方：A公司
乙方：B公司

第一条 标的物
本合同标的物为XXX产品，规格XXX，数量100台。

第二条 价款
总价款人民币100万元（含税）。

第三条 违约责任
违约方应支付违约金人民币10万元。
"""

user_context = {
    'party': '甲方',          # 代表方
    'position': '平等',       # 市场地位：强势/平等/弱势
    'history': '无',          # 过往交易
    'focus': '付款安全'       # 关注点
}

result = quick_review(
    contract_text=contract_text,
    contract_name='A公司B公司买卖合同',
    user_context=user_context,
    review_depth='standard'    # quick / standard / deep
)

print(f"识别类型: {result['analysis_result']['identified_type']}")
print(f"发现风险: {result['risk_report']['total_risks']}个")
print(f"法律意见书: {result['opinion_file']}")
```

**步骤 3：获取审核结果**

系统将输出：

- 📊 识别的合同类型
- 📋 发现的风险点数量和等级
- ✅ 法律审核意见书（自动生成）
- 📝 详细批注版合同

### 使用方式二：深度审核

**步骤 1：使用三观四步法深度审核**

```python
from main import advanced_review

# 三观四步法深度审核
result, opinion_file = advanced_review(
    contract_text=contract_text,
    contract_name='重要买卖合同',
    user_context=user_context,
    review_depth='standard'
)

# 查看智能评分
scoring = result['intelligent_scoring']
print(f"综合评分: {scoring['comprehensive_score']}/100")
print(f"风险等级: {scoring['risk_level']}")

print(f"\n各维度评分:")
for dim, score in scoring['dimension_scores'].items():
    print(f"  {dim}: {score}")

print(f"\n深度意见书: {opinion_file}")
```

**步骤 2：获取深度分析**

系统将输出：

- 📊 宏观/中观/微观三维度分析
- 📋 各维度评分（0-100）
- ✅ 深度法律审核意见书
- 💡 优化建议和修改方案

## ❓ 常见问题

### Q1：这个系统适合哪些人群使用？

**A**：

- 👨‍💼 **律师/法律顾问**：应急审查、重点分析、深度尽调
- 🏢 **企业法务**：日常审核、质量监控、培训工具
- 🎓 **法学教师/学生**：学习合同审核方法、熟悉实务
- 📋 **业务人员**：初步风险评估、理解合同条款

### Q2：系统生成的审核意见可以直接使用吗？

**A**：❌ **不可以！** AI 生成的内容必须经过专业法律人士审核：

- ⚠️ 核对法律引用是否正确
- ⚠️ 确认分析逻辑是否严谨
- ⚠️ 判断结论是否合理
- ⚠️ 调整具体建议和策略

**重要提示**：使用本系统产生的法律后果由使用者承担。

### Q3：三种审核模式有什么区别？

**A**：

- **快速审核**（5-10分钟）：识别重大风险，适合应急审查
- **标准审核**（30-60分钟）：全面风险评估，适合日常审核
- **深度审核**（1-2小时）：三观四步法完整分析，适合重大交易
